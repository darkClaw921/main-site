# Main Site - Clock Display

Веб-приложение на FastAPI для отображения текущего времени с кастомным шрифтом.

## Установка

```bash
uv pip install -e .
```

## Запуск

```bash
python run.py
```

Приложение будет доступно по адресу: http://localhost:8000

## Описание

Приложение отображает текущее время (часы:минуты:секунды:миллисекунды) с использованием кастомного шрифта Alteran.

### Особенности

- **Две темы оформления:**
  - Обычная тема: серые цифры на черном фоне
  - Тема "Матрица": зеленые цифры с эффектом свечения + анимация падающих символов
- **Переключение тем:** нажмите клавишу **ПРОБЕЛ** для смены темы
- Обновление времени каждые 10 миллисекунд
- Адаптивный дизайн для всех устройств

## Структура

- `app/` - основное приложение FastAPI
- `app/static/` - статические файлы (шрифты, стили, скрипты)
  - `alteran.ttf` - кастомный шрифт
  - `matrix-theme.css` - стили темы Матрицы
  - `matrix-effect.js` - анимация падающих символов
- `app/templates/` - HTML шаблоны

## Технологии

- FastAPI
- Uvicorn
- Loguru (логирование)
- Jinja2 (шаблоны)

Подробная документация архитектуры находится в файле `architecture.md`.
